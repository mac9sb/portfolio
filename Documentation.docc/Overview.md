# Portfolio

Welcome to **Portfolio**, a Swift package that powers a personal portfolio site using server-side Swift.

## Overview

This project focuses on a clean, static-friendly web experience built in Swift. It renders content using the `WebUI` and `WebUIMarkdown` packages and ships as an executable.

## Requirements

- Swift 6.2
- macOS 15+ (for development)

## Basic Setup

1. Clone the repository.
2. Resolve dependencies:
   - `swift package resolve`
3. Build and run:
   - `swift run`
