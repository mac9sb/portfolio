Building CLI tools with ArgumentParser

<div class="markdown-content"><h1>Building CLI Tools with ArgumentParser</h1><p>Swift’s ArgumentParser library makes building command-line tools straightforward and type-safe. This guide covers everything from basic commands to advanced patterns.</p><h2>Getting Started</h2><p>Add ArgumentParser to your package:</p><pre class="markdown-code-block language-swift"><code>// Package.swift dependencies: [ .package(url: &quot;https://github.com/apple/swift-argument-parser&quot;, from: &quot;1.3.0&quot;) ] </code></pre><h2>Basic Command</h2><p>A simple command with arguments:</p><pre class="markdown-code-block language-swift"><code>import ArgumentParser @main struct Greet: ParsableCommand { @Argument(help: &quot;The name to greet&quot;) var name: String @Option(name: .shortAndLong, help: &quot;Number of times to greet&quot;) var count: Int = 1 @Flag(name: .shortAndLong, help: &quot;Use formal greeting&quot;) var formal: Bool = false func run() throws { let greeting = formal ? &quot;Good day&quot; : &quot;Hello&quot; for _ in 0..&lt;count { print(&quot;\(greeting), \(name)!&quot;) } } } </code></pre><p>Usage:</p><pre class="markdown-code-block language-bash"><code>$ greet World --count 3 Hello, World! Hello, World! Hello, World! $ greet &quot;Dr. Smith&quot; --formal Good day, Dr. Smith! </code></pre><h2>Subcommands</h2><p>Organizing complex CLIs with subcommands:</p><pre class="markdown-code-block language-swift"><code>@main struct TodoCLI: ParsableCommand { static let configuration = CommandConfiguration( commandName: &quot;todo&quot;, abstract: &quot;A task management tool&quot;, subcommands: [Add.self, List.self, Complete.self], defaultSubcommand: List.self ) } extension TodoCLI { struct Add: ParsableCommand { static let configuration = CommandConfiguration( abstract: &quot;Add a new task&quot; ) @Argument(help: &quot;Task description&quot;) var description: String @Option(name: .shortAndLong, help: &quot;Priority level&quot;) var priority: Priority = .medium func run() throws { try TaskManager.add(description, priority: priority) print(&quot;Task added successfully&quot;) } } struct List: ParsableCommand { static let configuration = CommandConfiguration( abstract: &quot;List all tasks&quot; ) @Flag(name: .shortAndLong, help: &quot;Show completed tasks&quot;) var all: Bool = false func run() throws { let tasks = try TaskManager.list(includeCompleted: all) for task in tasks { print(task.formatted()) } } } } </code></pre><h2>Custom Types</h2><p>ArgumentParser works with custom types:</p><pre class="markdown-code-block language-swift"><code>enum Priority: String, ExpressibleByArgument, CaseIterable { case low, medium, high, urgent static var allValueStrings: [String] { allCases.map { $0.rawValue } } } struct DateArgument: ExpressibleByArgument { let date: Date init?(argument: String) { let formatter = DateFormatter() formatter.dateFormat = &quot;yyyy-MM-dd&quot; guard let date = formatter.date(from: argument) else { return nil } self.date = date } } </code></pre><h2>Validation</h2><p>Custom validation logic:</p><pre class="markdown-code-block language-swift"><code>struct CreateUser: ParsableCommand { @Option(help: &quot;User email&quot;) var email: String @Option(help: &quot;User age&quot;) var age: Int func validate() throws { guard email.contains(&quot;@&quot;) else { throw ValidationError(&quot;Invalid email format&quot;) } guard age &gt;= 13 else { throw ValidationError(&quot;Users must be at least 13 years old&quot;) } } } </code></pre><h2>Async Commands</h2><p>Support for async/await:</p><pre class="markdown-code-block language-swift"><code>struct FetchData: AsyncParsableCommand { @Option(help: &quot;API endpoint&quot;) var endpoint: String func run() async throws { let data = try await fetchFromAPI(endpoint) print(data.formatted()) } } </code></pre><h2>Best Practices</h2><ol><li><p><strong>Use descriptive help text</strong> for all arguments</p></li><li><p><strong>Provide sensible defaults</strong> where possible</p></li><li><p><strong>Validate early</strong> in the <code>validate()</code> method</p></li><li><p><strong>Structure with subcommands</strong> for complex tools</p></li><li><p><strong>Follow Unix conventions</strong> for options and flags</p></li></ol><h2>Conclusion</h2><p>ArgumentParser provides a robust foundation for building Swift CLI tools. Its type-safe approach catches errors at compile time and generates professional help text automatically.</p></div>
